Comece como quiser

Vá direto para a construção com o Bootstrap - use o CDN, instale-o por meio do gerenciador de pacotes ou baixe o código-fonte.

Leia os manual de instalação

Instalar via gerenciador de pacotes

Instale os arquivos Sass e JavaScript de origem do Bootstrap via npm, RubyGems, Composer ou Meteor. As instalações gerenciadas por pacote não incluem manual ou nossos scripts de compilação completos. Você também pode usar qualquer demonstração do nosso repositório de exemplos para iniciar rapidamente os projetos Bootstrap. 

npm install bootstrap@5.3.1
gem install bootstrap -v 5.3.1

Leia nossos manual de instalação para mais informações e gerenciadores de pacotes adicionais.

Incluir via CDN

Quando você só precisa incluir o CSS ou JS compilado do Bootstrap, você pode usar jsDelivr. Veja-o em ação com nosso simples começo rápido, ou navegue pelos exemplos para alavancar seu próximo projeto. Você também pode optar por incluir Popper e nosso JS separadamente. 

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.1/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-4bw+/aepP/YC94hEpVNVgiZdgIC5+VKNBQNGCHeKRQN+PtmoHDEXuppvnDJzQIu9" crossorigin="anonymous">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.1/dist/js/bootstrap.bundle.min.js" integrity="sha384-HwwvtgBNo3bZJJLYd8oVXjrBZt8cqVSpeBNS5n7C8IVInixGAoxmnlMuBnhbgrkm" crossorigin="anonymous"></script>

Personalize tudo com Sass

Bootstrap utiliza Sass para uma arquitetura modular e personalizável. Importe apenas os componentes necessários, habilite opções globais como gradientes e sombras e escreva seu próprio CSS com nossas variáveis, mapas, funções e mixins.

Saiba mais sobre personalização

Inclua todo o Sass do Bootstrap

Importe uma folha de estilo e comece a correr com todos os recursos do nosso CSS. 

// Substituições de variáveis primeiro
$primary: #900;
$enable-shadows: true;
$prefix: "mo-";

// Em seguida, importe o Bootstrap
@import "../node_modules/bootstrap/scss/bootstrap";

Saiba mais sobre o nosso opções globais Sass.

Inclua o que você precisa

A maneira mais fácil de personalizar o Bootstrap - inclua apenas o CSS necessário. 

// Funções primeiro.
@import "../node_modules/bootstrap/scss/functions";

// Variável substitui o segundo.
$primary: #900;
$enable-shadows: true;
$prefix: "mo-";

// Importações necessárias do Bootstrap.
@import "../node_modules/bootstrap/scss/variables";
@import "../node_modules/bootstrap/scss/variables-dark";
@import "../node_modules/bootstrap/scss/maps";
@import "../node_modules/bootstrap/scss/mixins";
@import "../node_modules/bootstrap/scss/root";

// Componentes opcionais.
@import "../node_modules/bootstrap/scss/utilities";
@import "../node_modules/bootstrap/scss/reboot";
@import "../node_modules/bootstrap/scss/containers";
@import "../node_modules/bootstrap/scss/grid";
@import "../node_modules/bootstrap/scss/helpers";
@import "../node_modules/bootstrap/scss/utilities/api";

Crie e estenda em tempo real com variáveis CSS.

O Bootstrap 5 está evoluindo a cada lançamento para melhor utilizar as variáveis CSS para estilos de temas globais, componentes individuais e até mesmo utilitários. Fornecemos dezenas de variáveis para cores, estilos de fonte e muito mais no nível :root para uso em qualquer lugar. Em componentes e utilitários, as variáveis CSS são definidas para a classe relevante e podem ser facilmente modificadas.

Saiba mais sobre variáveis CSS

Usando variáveis CSS

Use qualquer um de nossas variáveis global :root para escrever novos estilos. Variáveis CSS usam a sintaxe var(--bs-variableName) e pode ser herdado por elementos filhos. 

.component {
  color: var(--bs-gray-800);
  background-color: var(--bs-gray-100);
  border: 1px solid var(--bs-gray-200);
  border-radius: .25rem;
}

.component-header {
  color: var(--bs-purple);
}

Personalizando via variáveis CSS

Substitua as variáveis globais, de componente ou de classe de utilitário para personalizar o Bootstrap exatamente como você gosta. Não há necessidade de redeclarar cada regra, apenas um novo valor de variável. 

body {
  --bs-body-font-family: var(--bs-font-monospace);
  --bs-body-line-height: 1.4;
  --bs-body-bg: var(--bs-gray-100);
}

.table {
  --bs-table-color: var(--bs-gray-600);
  --bs-table-bg: var(--bs-gray-100);
  --bs-table-border-color: transparent;
}

Componentes, conheça a Utility API

Novidade no Bootstrap 5, nossos utilitários agora são gerados por nosso API do utilitário. Nós o construímos como um mapa Sass repleto de recursos que pode ser personalizado de forma rápida e fácil. Nunca foi tão fácil adicionar, remover ou modificar qualquer classe de utilitário. Torne os utilitários responsivos, adicione variantes de pseudoclasse e dê a eles nomes personalizados.

Personalize componentes rapidamente

Aplique qualquer uma de nossas classes de utilitários incluídas em nossos componentes para personalizar sua aparência, como no exemplo de navegação abaixo. Existem centenas de classes disponíveis, desde posicionamento e dimensionamento para cores e efeitos. Misture-os com substituições de variáveis CSS para obter ainda mais controle. 

<ul class="nav nav-pills nav-fill gap-2 p-1 small bg-primary rounded-5 shadow-sm" id="pillNav2" role="tablist" style="--bs-nav-link-color: var(--bs-white); --bs-nav-pills-link-active-color: var(--bs-primary); --bs-nav-pills-link-active-bg: var(--bs-white);">
  <li class="nav-item" role="presentation">
    <button class="nav-link active rounded-5" id="home-tab2" data-bs-toggle="tab" type="button" role="tab" aria-selected="true">Home</button>
  </li>
  <li class="nav-item" role="presentation">
    <button class="nav-link rounded-5" id="profile-tab2" data-bs-toggle="tab" type="button" role="tab" aria-selected="false">Profile</button>
  </li>
  <li class="nav-item" role="presentation">
    <button class="nav-link rounded-5" id="contact-tab2" data-bs-toggle="tab" type="button" role="tab" aria-selected="false">Contact</button>
  </li>
</ul>

Explorar componentes personalizados

Criar e estender utilitários

Use a API de utilitários do Bootstrap para modificar qualquer um de nossos utilitários incluídos ou criar seus próprios utilitários personalizados para qualquer projeto. Importe o Bootstrap primeiro e, em seguida, use as funções de mapa Sass para modificar, adicionar ou remover utilitários. 

@import "bootstrap/scss/bootstrap";

$utilities: map-merge(
  $utilities,
  (
    "cursor": (
      property: cursor,
      class: cursor,
      responsive: true,
      values: auto pointer grab,
    )
  )
);

Plugins JavaScript poderosos sem jQuery

Adicione elementos ocultos alternáveis, modais e menus fora da tela, popovers e dicas de ferramentas e muito mais - tudo sem jQuery. O JavaScript do Bootstrap é HTML-first, o que significa que a maioria dos plug-ins são adicionados com atributos data em seu HTML. Precisa de mais controle? Inclua plug-ins individuais programaticamente.

Saiba mais sobre Bootstrap JavaScript

API de atributos de dados

Por que escrever mais JavaScript quando você pode escrever HTML ? Quase todos os plug-ins JavaScript do Bootstrap apresentam uma API de dados de primeira classe, permitindo que você use JavaScript apenas adicionando atributos data. 

<div class="dropdown">
  <button class="btn btn-primary dropdown-toggle" type="button" data-bs-toggle="dropdown" aria-expanded="false">
    Dropdown
  </button>
  <ul class="dropdown-menu">
    <li><button class="dropdown-item" type="button">Dropdown item</button></li>
    <li><button class="dropdown-item" type="button">Dropdown item</button></li>
    <li><button class="dropdown-item" type="button">Dropdown item</button></li>
  </ul>
</div>

Aprender mais sobre nosso JavaScript como módulos e usando a API programática.

Conjunto abrangente de plugins

O Bootstrap apresenta uma dúzia de plug-ins que você pode inserir em qualquer projeto. Coloque-os todos de uma vez ou escolha apenas os que você precisa.

Alerta

Mostre e oculte mensagens de alerta para seus usuários.

Botão

Controle programaticamente o estado ativo dos botões.

Carousel

Adicione apresentações de slides a qualquer página, incluindo suporte para crossfade.

Collapse

Expanda e reduza áreas de conteúdo ou crie sanfonas.

Dropdown

Crie menus de links, ações, formulários e muito mais.

Modal

Adicione caixas de diálogo flexíveis e responsivas ao seu projeto.

Offcanvas

Crie e alterne barras laterais ocultas em qualquer página.

Popover

Crie sobreposições personalizadas. Construído em Popper.

Scrollspy

Atualize automaticamente os links de navegação ativos com base na rolagem da página.

Tab

Permitir que os componentes de navegação do Bootstrap alternem o conteúdo.

Toast

Mostre e oculte notificações para seus visitantes.

Tooltip

Substitua as dicas de ferramentas do navegador por outras personalizadas. Construído em Popper.

Personalize com Ícones Bootstrap

Ícones Bootstrap é uma biblioteca de ícones SVG de código aberto com mais de 1.800 glifos, com mais adicionados a cada lançamento. Eles são projetados para funcionar em qualquer projeto, quer você use o próprio Bootstrap ou não. Use-os como SVGs ou fontes de ícones - ambas as opções oferecem dimensionamento vetorial e fácil personalização via CSS.
