Usando bootstrap-modal

Chame o modal via javascript: 

$('#myModal').modal(options)

Opções

Nome Tipo Padrão Descrição
backdrop boolean true Inclui um elemento de modal-backdrop. Como alternativa, especifique static para um backdrop que não fecha o modal ao clicar.
keyboard boolean true Fecha o modal quando a tecla Escape é pressionada
show boolean true Mostra o modal quando inicializado.

Marcação

Você pode ativar modals em sua página facilmente, sem precisar escrever uma única linha de javascript. Basta definir data-toggle="modal" em um elemento controlador com um data-target="#foo" ou href="#foo" que corresponde a um id de elemento modal e, quando clicado, iniciará seu modal.

Além disso, para adicionar opções à sua instância modal, basta incluí-las como atributos de dados adicionais no elemento de controle ou na própria marcação modal. 

<a class="btn" data-toggle="modal" href="#myModal" >Launch Modal</a>
<div class="modal" id="myModal">
    <div class="modal-header">
        <a class="close" data-dismiss="modal">
            ×
        </a>

        <h3>
            Título modal
        </h3>
    </div>

    <div class="modal-body">
        <p>
            Um bom conteúdo...
        </p>
    </div>

    <div class="modal-footer">
        <a href="#" class="btn">
            Fechar
        </a>

        <a href="#" class="btn btn-primary">
            Salvar alterações
        </a>
    </div>
</div>
Atenção ! Se você quiser que seu modal entre e saia animado, basta adicionar uma classe .fade para o elemento .modal (consulte a demonstração para ver isso em ação) e inclua bootstrap-transition.js.

Métodos

.modal(options)

Ativa seu conteúdo como modal. Aceita um object de opções opcionais. 

$('#myModal').modal({
    keyboard: false
})

.modal('toggle')

Alterna manualmente um modal. 

$('#myModal').modal('toggle')

.modal('show')

Abre manualmente um modal. 

$('#myModal').modal('show')

.modal('hide')

Oculta manualmente um modal. 

$('#myModal').modal('hide')

Eventos

A classe modal do Bootstrap expõe alguns eventos para conectar-se à funcionalidade modal.

Evento Descrição
show Este evento é acionado imediatamente quando o método de instância show é chamado.
shown Este evento é acionado quando o modal fica visível para o usuário (aguardará a conclusão das transições CSS).
hide Este evento é enviado imediatamente quando o método de instância hide tem sido chamado.
hidden Este evento é acionado quando o modal termina de ser ocultado do usuário (aguardará a conclusão das transições CSS). 
$('#myModal').on('hidden', function ()
{
    //
    // Faça alguma coisa...
    //
})

Ativa um elemento de tab e um contêiner de conteúdo. A tab deve ter um `data-target` ou um `href` direcionado a um nó de contêiner no dom. 

<ul class="nav nav-tabs">
    <li class="active">
        <a href="#home">
            Casa
        </a>
    </li>

    <li>
        <a href="#profile">
            Perfil
        </a>
    </li>

    <li>
        <a href="#messages">
            Mensagens
        </a>
    </li>

    </li>

        ...
    </div>

    <div class="tab-pane" id="profile">
        ...
    </div>

    <div class="tab-pane" id="messages">
        ...
    </div>

    <div class="tab-pane" id="settings">
        ...
    </div>
</div>

<script>
    $(function ()
    {
        $('.tabs a:last').tab('show');
    });
</script>

Eventos

Evento Descrição
show Este evento é acionado na exibição da tab, mas antes da exibição da nova tab. Use event.target e event.relatedTarget para direcionar a tab ativa e a tab ativa anterior (se disponível), respectivamente.
shown Este evento é acionado na exibição da tab após a exibição de uma tab. Use event.target e event.relatedTarget para direcionar a tab ativa e a tab ativa anterior (se disponível), respectivamente. 
$('a[data-toggle="tab"]').on('shown', function (e)
{
    //
    // Tab ativada.
    //
    e.target;

    //
    // Tab anterior.
    //
    e.relatedTarget;
});

Usando bootstrap-tooltip.js

Acione o tooltip via javascript: 

$('#example').tooltip(options)

Opções

Nome Tipo Padrão Descrição
animation boolean true aplique uma transição de fade css ao tooltip
placement string|function 'top' como posicionar o tooltip - top | bottom | left | right
selector string false Se um seletor for fornecido, os objetos de tooltip serão delegações aos destinos especificados.
title string | function '' valor padrão do título se a tag `title` não estiver presente

Atenção ! Opções para tooltips individuais podem ser especificadas alternativamente através do uso de atributos de dados.

Marcação

<a href="#" rel="tooltip" title="first tooltip">hover over me</a>

Métodos

.tooltip('hide')

Oculta o tooltip de um elemento. 

$('#element').tooltip('hide')

.tooltip('toggle')

Alterna o tooltip de um elemento. 

$('#element').tooltip('toggle')

Usando bootstrap-collapse.js

Habilitar via javascript: 

$(".collapse").collapse()

Opções

Nome Tipo Padrão Descrição
parent selector false Se for seletor, todos os elementos recolhíveis sob a camada mais alta especificado serão fechados quando este item recolhível for mostrado. (semelhante ao comportamento tradicional do accordion)
toggle boolean true Alterna o elemento recolhível na chamada

Marcação

Basta adicionar data-toggle="collapse" e um data-target para elemento para atribuir automaticamente o controle de um elemento recolhível. O atributo data-target aceita um seletor css para aplicar o recolhimento. Certifique-se de adicionar a classe collapse ao elemento dobrável. Se você quiser que ele seja aberto por padrão, adicione a classe in adicional. 

<button class="btn btn-danger" data-toggle="collapse" data-target="#demo">
    collapse simples
</button>

<div id="demo" class="collapse in">
    ...
</div>
Atenção ! Para adicionar gerenciamento de grupo tipo accordion a um controle collapse, adicione o atributo data data-parent="#selector". Consulte a demonstração para ver isso em ação.

Métodos

.collapse(options)

Ativa seu conteúdo como um elemento recolhível. Aceita opções opcionais object. 

$('#myCollapsible').collapse({
    toggle: false
});

.collapse('toggle')

Alterna um elemento recolhível para mostrado ou oculto.

.collapse('show')

Mostra um elemento collapse.

.collapse('hide')

Oculta um elemento collapse.

Eventos

A classe de collapse do Bootstrap expõe alguns eventos para conectar-se à funcionalidade de collapse.

Evento Descrição
show Este evento é acionado imediatamente quando o método de instância show é chamado.
shown Este evento é acionado quando um elemento de collapse se torna visível para o usuário (aguardará a conclusão das transições CSS).
hide Este evento é enviado imediatamente quando o método hide tem sido chamado.
hidden Este evento é enviado quando um elemento de recolhimento foi ocultado do usuário (aguardará a conclusão das transições CSS). 
$('#myCollapsible').on('hidden', function ()
{
    //
    // Faça alguma coisa...
    //
});

Usando bootstrap-carousel.js

Chamar via javascript: 

$('.carousel').carousel()

Opções

Nome Tipo Padrão Descrição
interval number 5000 A quantidade de tempo de atraso entre o ciclo automático de um item.
pause string "hover" Pausa o ciclo do carrossel ao entrar com o mouse e retoma o ciclo do carrossel ao sair do mouse.

Marcação

Os atributos de dados são usados para os controles anteriores e seguintes. Confira o exemplo de marcação abaixo. 

<div id="myCarousel" class="carousel">
    <!--
      - Itens do carrossel.
     -->
    <div class="carousel-inner">
        <div class="active item">
            ...
        </div>

        <div class="item">
            ...
        </div>

        <div class="item">
            ...
        </div>
    </div>

    <!--
      - Navegação em carrossel.
     -->
    <a class="carousel-control left" href="#myCarousel" data-slide="prev">
        &lsaquo;
    </a>

    <a class="carousel-control right" href="#myCarousel" data-slide="next">
        &rsaquo;
    </a>
</div>

Métodos

.carousel(options)

Inicializa o carrossel com opções opcionais object e começa a percorrer os itens. 

$('.carousel').carousel({
    interval: 2000
});

.carousel('cycle')

Percorre os itens do carrossel da esquerda para a direita.

.carousel('pause')

Impede que o carrossel percorra os itens.

.carousel(number)

Percorre o carrossel para um quadro específico (baseado em 0, semelhante a um vetor).

.carousel('prev')

Volta para o item anterior.

.carousel('next')

Passa para o próximo item.

Eventos

A classe carrossel do Bootstrap expõe dois eventos para conexão com a funcionalidade do carrossel.

Evento Descrição
slide Este evento é acionado imediatamente quando o método de instância slide é chamado.
slid Este evento é acionado quando o carrossel conclui a transição de slides.

Usando bootstrap-typeahead.js

Chamar o typeahead via javascript: 

$('.typeahead').typeahead()

Opções

Nome Tipo Padrão Descrição
source array [ ] A fonte de dados a ser consultada.
items number 8 O número máximo de itens a serem exibidos no menu dropdown.
matcher function case insensitive O método usado para determinar se uma consulta corresponde a um item. Aceita um único argumento, o item contra o qual testar a consulta. Acesse a consulta atual com this.query. Retornar um booleano true se a consulta for uma correspondência.
sorter function correspondência exata,
case sensitive,
case insensitive 		Método usado para classificar os resultados do preenchimento automático. Aceita um único argumento items e tem o escopo da instância de digitação antecipada. Faça referência à consulta atual com this.query.
highlighter function destaca todas as correspondências padrão Método usado para destacar resultados de preenchimento automático. Aceita um único argumento item e tem o escopo da instância de digitação antecipada. Deve retornar html.

Marcação

Adicione atributos de dados para registrar um elemento com funcionalidade de digitação antecipada. 

<input type="text" data-provide="typeahead">

Métodos

.typeahead(options)

Inicializa uma entrada com digitação antecipada.
